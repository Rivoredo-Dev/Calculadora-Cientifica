# 🧮 Calculadora Científica - Java

Projeto de uma calculadora modular desenvolvida em Java, focada na aplicação prática de **Programação Orientada a Objetos (POO)**.

### 🚀 Funcionalidades
* **Modo Básico:** Operações aritméticas fundamentais.
* **Modo Científico:** Funções avançadas como fatorial, inversão e outras operações modulares.
* **Arquitetura:** Design extensível, facilitando a adição de novas funções matemáticas sem alterar o núcleo do sistema.

### 🛠️ Tecnologias
* Java (JDK 11+)
* Princípios de Clean Code e POO

### 🏁 Como Executar
1. Certifique-se de ter o JDK instalado.
2. Compile o projeto:
   ```bash
   javac AppCalc.java
